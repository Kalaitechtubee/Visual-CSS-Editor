# Changelog

All notable changes to Visual CSS Editor will be documented in this file.

## [1.0.0] - 2024-12-12

### Added
- Initial release
- Element inspection with hover highlighting
- Visual CSS editing:
  - Typography (font family, size, weight, line height, color)
  - Background (color, opacity, blur)
  - Appearance (opacity, rotation, border radius, blend mode)
  - Spacing (margin, padding with visual controls)
  - Border (width, style, color per side)
  - Layout (display, flex/grid controls, gap)
  - Box Shadow (multiple shadows with full controls)
- Preset system with save/load/delete
- Changes tracking with undo/redo
- CSS export (copy, download)
- Pro features:
  - Apply to similar elements
  - Screenshot export (PNG)
  - AI code generation (Groq API)
  - Framework conversion (React, Vue, Angular)
- Dark mode UI
- Keyboard shortcuts
